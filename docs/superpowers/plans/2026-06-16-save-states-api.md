# Save States — API Foundation Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the `save_states` data model + REST API (personal upsert, team-digest edit, recall bundle, list, delete) that the EOD job, CLI, and dashboard plans depend on.

**Architecture:** One `save_states` table with a `kind` discriminator (`personal` | `team_daily` | `team_weekend`). Personal rows are per (project, user); team rows are per (project, kind) with a null `user_id`. The controller is org-scoped (codebase convention) with a required `project_id` param. Content is sanitized at the write boundary. Recall resolves the weekend-vs-weekday injection rule server-side. This plan is **API-only** — the EOD job (Plan 2), CLI (Plan 3), and dashboard (Plan 4) build on these endpoints.

**Tech Stack:** Rails 8.1, PostgreSQL 17 (partial unique indexes, `gen_random_uuid()`), Pundit, `acts_as_paranoid`, `paper_trail`, Minitest fixtures.

---

## Design reconciliations vs. the spec (`2026-06-15-session-save-states-design.md`)

- **Routes are org-scoped flat** (`/:org/save_states`, `project_id` as a param), matching `artifacts`/`inbound_messages`, not nested under `/projects/:id`.
- **No project-lead role exists.** Team-digest edit/delete is gated on `manager?` (owner/admin/manager). The spec's "admin/project-lead" maps to this.
- **Team-row uniqueness uses two partial unique indexes** (null `user_id` would defeat a plain composite unique index in Postgres).
- Team digests are **created by the Plan-2 job**; this plan creates them directly in tests to exercise recall/edit/delete.

class CreateSaveStates < ActiveRecord::Migration[8.1]
  def change
    create_table :save_states do |t|
      t.uuid :external_id, default: -> { "gen_random_uuid()" }, null: false
      t.references :organization, null: false, foreign_key: true
      t.references :project, null: false, foreign_key: true

      # personal | team_daily | team_weekend
      t.string :kind, null: false

      # set for personal (owner); null for team_*
      t.references :user, null: true, foreign_key: true
      # last human to save/edit (attribution); null when system-generated
      t.references :created_by, null: true, foreign_key: { to_table: :users }
      # team_* only: lead/admin who reviewed
      t.references :reviewed_by, null: true, foreign_key: { to_table: :users }
      t.datetime :reviewed_at

      # personal provenance; nullify so a deleted session leaves the state intact
      t.references :source_llm_session, null: true,
                   foreign_key: { to_table: :llm_sessions, on_delete: :nullify }

      # team_* only
      t.date :digest_date
      t.date :period_start # team_weekend only

      t.text :handoff, null: false
      t.jsonb :metadata, null: false, default: {}
      t.datetime :deleted_at

      t.timestamps

      t.index :external_id, unique: true
      t.index :deleted_at

      # One personal row per (project, user).
      t.index [ :project_id, :user_id ], unique: true,
              where: "kind = 'personal' AND deleted_at IS NULL",
              name: "idx_save_states_one_personal_per_user_project"

      # One team_daily and one team_weekend row per project.
      t.index [ :project_id, :kind ], unique: true,
              where: "user_id IS NULL AND deleted_at IS NULL",
              name: "idx_save_states_one_team_row_per_project_kind"
    end
  end
end

# api/app/models/save_state.rb
class SaveState < ApplicationRecord
  acts_as_paranoid

  has_paper_trail on: [ :update, :destroy ],
                  meta: { organization_id: :organization_id },
                  versions: { class_name: "Version" }

  belongs_to :organization
  belongs_to :project
  belongs_to :user, optional: true
  belongs_to :created_by, class_name: "User", optional: true
  belongs_to :reviewed_by, class_name: "User", optional: true
  belongs_to :source_llm_session, class_name: "LlmSession", optional: true

  PERSONAL = "personal".freeze
  TEAM_DAILY = "team_daily".freeze
  TEAM_WEEKEND = "team_weekend".freeze
  KINDS = [ PERSONAL, TEAM_DAILY, TEAM_WEEKEND ].freeze
  TEAM_KINDS = [ TEAM_DAILY, TEAM_WEEKEND ].freeze
  MAX_HANDOFF_BYTES = 16_384

  validates :kind, inclusion: { in: KINDS }
  validates :handoff, presence: true
  validate :handoff_within_byte_limit
  validate :one_personal_per_project_user, if: :personal?
  validate :one_team_row_per_project, if: :team?

  scope :personal_states, -> { where(kind: PERSONAL) }
  scope :team_states, -> { where(kind: TEAM_KINDS) }

  def personal?
    kind == PERSONAL
  end

  def team?
    TEAM_KINDS.include?(kind)
  end

  def to_param
    external_id
  end

  # Mirrors SessionVectorizationService#vectorizable? sensitivity gate.
  def self.session_recallable?(session)
    return true if session.nil?
    return false if session.sensitive?
    return false if session.repository && !session.repository.public_visibility?

    true
  end

  # Resolves which layers a new session should be injected with.
  # Weekend is included only when it is more recent than the weekday digest,
  # so Monday gets Friday + weekend, but Tuesday onward drops the stale weekend.
  def self.recall_bundle(project:, user:)
    personal = find_by(project: project, kind: PERSONAL, user_id: user.id)
    daily = find_by(project: project, kind: TEAM_DAILY)
    weekend = find_by(project: project, kind: TEAM_WEEKEND)

    include_weekend = weekend.present? && (
      daily.nil? || daily.digest_date.nil? ||
      (weekend.digest_date.present? && weekend.digest_date > daily.digest_date)
    )

    { personal: personal, team_daily: daily, team_weekend: include_weekend ? weekend : nil }
  end

  private

  def handoff_within_byte_limit
    return if handoff.blank?

    errors.add(:handoff, "is too long (16 KB max)") if handoff.bytesize > MAX_HANDOFF_BYTES
  end

  def one_personal_per_project_user
    scope = SaveState.where(project_id: project_id, kind: PERSONAL, user_id: user_id)
    scope = scope.where.not(id: id) if persisted?
    errors.add(:user_id, "already has a personal save state for this project") if scope.exists?
  end

  def one_team_row_per_project
    scope = SaveState.where(project_id: project_id, kind: kind)
    scope = scope.where.not(id: id) if persisted?
    errors.add(:kind, "already exists for this project") if scope.exists?
  end
end
```

# api/app/policies/save_state_policy.rb
class SaveStatePolicy < ApplicationPolicy
  class Scope < ApplicationPolicy::Scope
    def resolve
      organization_scoped(scope)
    end
  end

  # Any member can list and recall.
  def index?
    class_or_member? { viewer? }
  end

  def recall?
    index?
  end

  # Saving a personal handoff is a "write" — developer and up.
  def personal?
    class_or_member? { developer? }
  end

  # Editing/deleting a team digest is a leadership action.
  # (No project-lead role exists; manager? = owner/admin/manager.)
  def update_team?
    crewkit_staff? || (belongs_to_organization? && manager?)
  end

  def destroy?
    return false unless belongs_to_organization? || crewkit_staff?
    return true if crewkit_staff?

    if record.personal?
      record.user_id == current_user&.id || manager?
    else
      manager?
    end
  end

  private

  def class_or_member?
    if record.is_a?(Class)
      current_organization.present? && yield
    else
      belongs_to_organization? && yield
    end
  end
end
```

> `current_user`, `crewkit_staff?`, `belongs_to_organization?`, and the role predicates come from `ApplicationPolicy` (see `api/app/policies/application_policy.rb`). Verify `current_user` is exposed there; if the base exposes the user as `user`, replace `current_user` with `user`.

# api/app/serializers/save_state_serializer.rb
class SaveStateSerializer < BaseSerializer
  attributes :id, :kind, :title, :handoff,
             :digest_date, :period_start,
             :saved_by, :reviewed_by, :reviewed_at,
             :source_session_id, :created_at, :updated_at

  link :self do |state|
    "/api/v1/#{state.organization.external_id}/save_states/#{state.external_id}"
  end

  private

  def id
    record.external_id
  end

  # Personal rows are titled by the saver; team rows by their kind + date.
  def title
    if record.personal?
      "Personal save state"
    elsif record.kind == SaveState::TEAM_WEEKEND
      "Team weekend digest"
    else
      "Team daily digest"
    end
  end

  def saved_by
    user = record.created_by || record.user
    return nil unless user

    { id: user.external_id, email: user.email, name: user.name }
  end

  def reviewed_by
    user = record.reviewed_by
    return nil unless user

    { id: user.external_id, email: user.email, name: user.name }
  end

  def source_session_id
    record.source_llm_session&.external_id
  end

  def handoff
    if context[:collection]
      record.handoff&.truncate(500)
    else
      record.handoff
    end
  end

  def self.serialize_collection(records, context = {})
    enriched = context.merge(collection: true)
    {
      data: records.map { |r| new(r, enriched).serialize[:data] },
      _links: collection_links(records, enriched),
      meta: collection_meta(records, enriched)
    }.compact
  end
end

# api/app/controllers/api/v1/save_states_controller.rb
module Api
  module V1
    class SaveStatesController < BaseController
      before_action :set_project, except: :destroy

      # GET /:org_id/save_states?project_id=...
      def index
        authorize SaveState

        states = policy_scope(SaveState)
          .where(organization: current_organization, project: @project)
          .where("kind != ? OR user_id = ?", SaveState::PERSONAL, current_user.id)
          .order(updated_at: :desc)

        render json: { data: states.map { |s| serialize(s, collection: true) } }
      end

      # GET /:org_id/save_states/recall?project_id=...
      def recall
        authorize SaveState, :recall?

        bundle = SaveState.recall_bundle(project: @project, user: current_user)
        render json: {
          data: bundle.transform_values { |s| s && serialize(s) }
        }
      end

      # PUT /:org_id/save_states/personal  { project_id, handoff, source_session_id? }
      def personal
        authorize SaveState, :personal?

        handoff = params[:handoff].to_s
        if handoff.bytesize > SaveState::MAX_HANDOFF_BYTES
          return render_error(:unprocessable_entity,
            "handoff exceeds the #{SaveState::MAX_HANDOFF_BYTES}-byte limit")
        end

        session = lookup_session(params[:source_session_id])
        unless SaveState.session_recallable?(session)
          return render_error(:unprocessable_entity,
            "source session is sensitive or private and cannot be saved")
        end

        state = current_organization.save_states.find_or_initialize_by(
          project: @project, kind: SaveState::PERSONAL, user_id: current_user.id
        )
        state.assign_attributes(
          handoff: ContentSanitizer.clean(handoff),
          created_by: current_user,
          source_llm_session: session
        )

        return render_validation_error(state) unless state.save

        render json: serialize(state)
      end

      # PATCH /:org_id/save_states/team/:kind  { project_id, handoff?, title? }
      def update_team
        kind = params[:kind].to_s
        unless SaveState::TEAM_KINDS.include?(kind)
          return render_error(:unprocessable_entity, "kind must be team_daily or team_weekend")
        end

        state = current_organization.save_states.find_by!(project: @project, kind: kind)
        authorize state, :update_team?

        state.handoff = ContentSanitizer.clean(params[:handoff].to_s) if params.key?(:handoff)
        state.reviewed_by = current_user
        state.reviewed_at = Time.current

        return render_validation_error(state) unless state.save

        render json: serialize(state)
      end

      # DELETE /:org_id/save_states/:external_id
      def destroy
        state = policy_scope(SaveState)
          .where(organization: current_organization)
          .find_by!(external_id: params[:external_id])
        authorize state, :destroy?
        state.destroy!
        head :no_content
      end

      private

      def set_project
        @project = current_organization.projects.find_by!(external_id: params[:project_id])
      end

      def lookup_session(external_id)
        return nil if external_id.blank?

        current_organization.llm_sessions.find_by(external_id: external_id)
      end

      def serialize(state, collection: false)
        SaveStateSerializer.new(state, collection: collection).serialize[:data]
      end
    end
  end
end
